# hid-switch2-dkms

a dkms module that adds switch 2 controler support (usb) to kernels 6.19


# Install

## Arch Linux

Use an aur helper to instal hid-switch2-dkms

example:

``` yay -S hid-switch2-dkms ```


## Other Distros

Download the source and install manually using dkms
